# DocuScript AI 🎬
### AI-Powered Cinematic Documentary Script Generator

**DocuScript AI** is a premium web application designed for filmmakers, documentary writers, and storytellers. Leveraging the power of Google's Gemini AI, it generates high-end, cinematic scripts tailored to specific director notes, themes, and visual styles.

## ✨ Features

- **Cinematic Script Generation**: Powered by `gemini-1.5-flash`, producing 8-10 scene structured screenplays.
- **Director-Driven Tailoring**: Customize generation with Title, Category, Tone, Key Figures, and Visual Style.
- **Secure Authentication**: User accounts and profile management via Firebase.
- **Premium UI/UX**: Futuristic "glassmorphism" aesthetic with real-time "Film Grain" effects and smooth transitions.
- **Export Ready**: Instantly download generated scripts as clean text files for further editing.
- **Dynamic Routing**: Smooth Single Page Application (SPA) experience.

## 🛠️ Tech Stack

- **Frontend**: Vanilla JavaScript (ES6+), HTML5, Tailwind CSS
- **AI Engine**: Google Gemini API (1.5 Flash)
- **Backend / Services**: 
  - Firebase Authentication
  - Firestore (Metadata Storage)
  - Firebase Analytics
- **Typography**: Inter & Material Symbols

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Abhinavmattam/Ai-powered-script-generator.git
   ```
2. **Open `index.html`**: Simply serve the project folder using any local development server (like VS Code Live Server).

## ⚙️ Configuration

- **Firebase**: Update your credentials in `js/firebase.js`.
- **Gemini API**: Configure your API key in `js/gemini.js`.
